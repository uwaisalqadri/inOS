## <a name="title"></a> ⚙️ inOS

App for Functionality Testing using **`CoreBluetooth`**, **`CoreLocation`**, **`CoreNFC`**, **`AVFoundation`**, **`CoreMotion`**, **`LocalAuthentication`**, **`AudioToolbox`**, **`Speech`**, **`UIDevice`**, **`Network`**, **`LocalAuthentication`**, **`HOST_CPU_LOAD_INFO`**, **`NSTemporaryDirectory`**. 
_______________________
All the important code is stored inside [inCore](https://github.com/uwaisalqadri/inCore) to maintain confidentiality. For a detailed implementation of each function, please refer to _[Accessing iPhone Hardware for Device Functionality Testing: Lessons Learned](https://medium.com/@uwaisalqadri/accessing-iphone-hardware-for-device-functionality-testing-lessons-learned-5d81676082d8)_
